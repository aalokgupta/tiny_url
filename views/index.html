API Basejump: URL Shortner microscervice

User Stroies:
  1. I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  2. IWhen I visit that shortened URL, it will redirect me to my original link.

Example Usage:

  • https://tranquil-salt.glitch.me/unix-timestamp/september 19 2017
  • https://tranquil-salt.glitch.me/unix-timestamp/1505865600

Example output:
  • { "unix": 1505865600, "natural": "Wed Sep 20 2017" }

Will Redirect to:
  • www.google.com
